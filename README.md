# ArduinoBluetoothLED
Arduino Control LED via Bluetooth Module ( HC-06 )

Hardware
1. Arduino UNO Board or compatible Board
2. Serial Bluetooth Module. ( HC-06 )
3. LED with Resistor 300 - 1K ohm x 8
